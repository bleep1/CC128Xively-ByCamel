getting it going...
====================
1) git clone https://github.com/bleep1/CC128Xively-UsingCamel.git
2) on jbds:
   import maven project


3) To build this project use

    mvn install

To run this project with Maven use

    mvn camel:run

For more help see the Apache Camel documentation

    http://camel.apache.org/
